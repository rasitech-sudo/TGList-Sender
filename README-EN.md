## **What is TGList Sender?**
TGList Sender is your tool to send messages to a telegram chat ID automatically via telegram bot.

## **Feature**

 - [x] Import List
 - [x] Send Message
 - [x] Send Photo
 - [x] Send Video
 - [x] Send Document
 - [x] Send Audio
 - [x] Add Description
 - [x] Change delivery rate with Interval
